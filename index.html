2024.12.19.(목) 기준

게임 소개

최적의 경로를 찾아가는 전략적 사고력 게임!

알고리즘 사고력 향상

최단 경로를 찾아가는 과정에서 알고리즘적 사고력이 향상됩니다.

지식 연결성 강화

서로 다른 주제 간의 연결고리를 찾으며 지식의 확장성을 경험합니다.

협동과 경쟁

다른 플레이어들과의 경쟁을 통해 더 효율적인 경로를 발견합니다.

게임 규칙

  • 시작값과 끝값은 나무위키에 등록된 단어로 합니다
  • Control + F는 서로 양 선수 합의하에 사용이 가능합니다
  • 페이지 이동수를 1트(횟수)로 계산합니다
  • 동일한 페이지 이동수를 보유한다면 먼저 도착한 사람이 승리합니다
